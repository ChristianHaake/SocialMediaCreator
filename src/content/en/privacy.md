## Local processing

Text, metrics, selected images, and imported configuration files are processed
only in the browser. The application has no backend and does not transmit this
content to the operator.

Image and configuration exports are created locally. Images are explicitly not
included in configuration files.

## Hosting and connection data

The website is delivered through Cloudflare Workers. When a website is
requested, technically necessary connection data may be processed, including
IP address, time, requested resource, browser information, and security
metadata.

This processing supports secure and reliable delivery. Where applicable, the
legal basis is Art. 6(1)(f) GDPR. Cloudflare provides further information in
its [privacy policy](https://www.cloudflare.com/privacypolicy/).

## Cookies, analytics, and external content

The application itself uses no analytics, advertising, or personalization
services and loads no external fonts or media. Cloudflare may use technically
necessary cookies when security measures are active.

## Retention

The application does not store user content on a server. Retention periods for
technical connection and security data are determined by the hosting provider
or responsible party.
